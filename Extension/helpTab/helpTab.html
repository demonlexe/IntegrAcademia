Help with sk.edge

This tool was first designed at HackUTD IX for students at the University of Texas at Dallas.

It was created by Bradley Johnson, Adam Szumski, and Alexis Kaufman.

*We recommend that you pin our extension for easy access.

FAQ :

  1. Open up Orion, then log into schedule planner.
  2. Click "Options" next to the course you want to know more about.
  3. Our extension will automatically insert the Professor name into the page.
  4. A "!" will pop-up next to the sk.edge extension icon in your browser's toolbar.
  5. Click on the icon!

  • Nebula API to access grade distributions.
  • Modified version of a RateMyProfessor API to obtain professor ratings.

We hope to be able to extend this project to other universities!

Please contact the developers and we would be happy to work with developers from other schools to implement this tool for you!
